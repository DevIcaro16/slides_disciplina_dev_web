TASK: USABILIDADE

Exibir alerta de confirmação ao tentar excluir um cliente

Qual foi o problema?

Descrição do Problema

Na tela de Clientes, ao clicar no botão "Excluir", o registro era removido imediatamente. O correto seria ter uma confirmação para evitar que o usuário excluisse o cliente de forma acidental.

Impacto

A falta de confirmação antes da exclusão de registros apresenta um grande risco para a integridade dos dados em qualquer sistema. A implementação de um modal de confirmação melhoraria a experiência do usuário, e também serveria como uma salvaguarda contra exclusões acidentais, oque reduzeria a necessidade de recuperação de dados e mantendo de forma positiva a confiabilidade, a visão e a opinião do usuário para com o nosso sistema .

Explicação da Solução

Implementação do Modal de Confirmação de Exclusão

Foi implementado um modal de confirmação de exclusão utilizando o framework frontend Bootstrap 5 para melhorar a usabilidade do sistema, evitando exclusões acidentais de registros.

Como foi resolvido? (Parte 1)

JavaScript - Implementação
                            
                                // JavaScript
                                document.addEventListener('DOMContentLoaded', function () {
                                    const deleteBtns = document.querySelectorAll('.btn-danger.btn-sm');
                                    deleteBtns.forEach(btn => {
                                        btn.setAttribute('data-bs-toggle', 'modal');
                                        btn.setAttribute('data-bs-target', '#modalDelete');
                                    });
                                });

TASK: DESIGN

Criar dashboard inicial com indicadores visuais

Qual foi o problema?

Descrição do Problema

O sistema não apresenta um painel na tela inicial descrevendo os principais números a respeito de: total de vendas do mês, produtos em baixo estoque, quantidade de clientes, etc.

Impacto

A maior parte de grandes sistemas possuem uma parte destinada a apresentação de dados/estatisticas com gráficos para a visualização de importantes informações que o cliente deseja ter entendimento. com um dashboard, nosso sistema ficará mais robusto, terá um design melhor e impressionará nossos atuais clientes e chamará a atenção de futuros clientes

Explicação da Solução

Implementação dos Gráficos

Foi utilizado a biblioteca Chart.JS para a elaboração de gráficos de diferentes tipos (Barras Horizontais, Barras Verticais, Pizza e etc), proporcionando uma visualização clara e interativa dos dados do sistema.
